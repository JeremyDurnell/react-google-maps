# Pan By

Changes the center of the map by the given distance in pixels.

If the distance is less than both the width and height of the map, the transition will be smoothly animated.

**Google Maps Docs:**

- [google.maps.Map](https://developers.google.com/maps/documentation/javascript/reference#Map)

<Playground>
  <TransitionExample
    zoom={8}
    center={{ lat: 36, lng: -122 }}
    initialState={{ key: "initial", offset: { x: 0, y: 0 } }}
    onAnimateClick={({ setState }) => {
      setState({
        key: Math.random(), // Generate new key to force animation with same offset.
        offset: { x: 100, y: 100 },
      });
    }}
    render={({ state }) => (
      <>
        <PanBy key={state.key} offset={state.offset} />
        <Marker label="C" position={{ lat: 36, lng: -122 }} />
      </>
    )}
  />
</Playground>
